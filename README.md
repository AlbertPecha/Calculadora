# Calculadora Interactiva en Python
Este proyecto consiste en una calculadora funcional de línea de comandos desarrollada en Python. 
Está diseñada bajo un enfoque de simplicidad y eficiencia, permitiendo al usuario realizar operaciones aritméticas básicas de forma intuitiva.
# Características Principales
Menú Interactivo: Presenta una interfaz limpia con cinco opciones numeradas (1-5).

# Operaciones Soportadas:

1. Suma: Cálculo de dos valores.

2. Resta: Diferencia entre números.

3. Multiplicación: Producto de los factores ingresados.

4. División: Cociente automático de los valores.

5. Salir: Opción para cerrar el programa de forma segura.

Automatización de Cálculos: El usuario solo necesita seleccionar la operación e ingresar los datos; el sistema procesa y muestra el resultado de inmediato.

# Detalles Técnicos
Lenguaje: Python 3.

Lógica de Control: Utiliza estructuras condicionales (if-elif-else) para gestionar las decisiones del usuario y un bucle (posiblemente while) para mantener la interfaz activa hasta que se decida salir.

Entrada de Datos: Gestión dinámica de números mediante la función input().

# Nota del autor: Este software destaca por su facilidad de uso, eliminando pasos innecesarios y enfocándose en la resolución directa de problemas matemáticos básicos.
